# **App Name**: FestConnect

## Core Features:

- Landing Page: Visually appealing landing page with hero section, features overview, and clear calls to action for registration and login.
- User Authentication: Secure user registration and login system with role-based access control (student, organizer, admin) using Firebase Authentication.
- Event Discovery and Registration: Browse events, view details, and register for events.
- Event Creation and Management: Organizers can create, edit, and manage event details including descriptions, schedules, and participant limits using a Firestore database.
- Role-Based Dashboards: Custom dashboards for students, organizers, and admins, providing relevant information and functionalities.
- Push Notifications: Push notifications to inform users about new events, registration confirmations, and event updates using Firebase Cloud Messaging.
- AI-Powered Event Recommendations: Suggest relevant events to users based on their profiles and past activity, using a tool that connects to Firestore.

## Style Guidelines:

- Primary color: Indigo (#6366f1) to convey a sense of trust and innovation.
- Background color: Very light lavender (#F5F3FF) for a clean, modern look.
- Accent color: Amber (#F59E0B) for calls to action and highlights, providing warmth and energy.
- Headline font: 'Poppins', a geometric sans-serif, for a precise, contemporary, fashionable feel.
- Body font: 'Inter', a grotesque-style sans-serif, offering a modern, machined, neutral aesthetic for readability.
- Lucide React icons: Consistent, modern icons from Lucide to represent various actions and categories.
- Modern UI: Incorporate gradients, subtle shadows, and a clean layout with ample white space to create a visually appealing and user-friendly interface, in line with contemporary design trends.
- Framer Motion animations: Smooth transitions and animations using Framer Motion to enhance user engagement and provide visual feedback.
